Sinthetic Labs | Open source security tools.


--------------------------

Sinthetic Labs creates and contributes to various open source projects we use to perform our research and work. We feel it's important to contribute back to the community, and to show our appreciation to the other firms and individuals who give their time to develop the tools we use. All of our projects can be found below and on Github. If you use any of our tools it would be greatly appreciated if you give us feedback, contribute patches, or write about your work for the community to see. Tweet us on Twitter if there's something cool you'd like us to check out.

-- Open Source Media

Open Source Media is an advanced OSINT desktop application for finding and studying online media.

-- Instagram Pushpin Module

"Recon-ng is a full-featured Web Reconnaissance framework written in Python. Complete with independent modules, database interaction, built in convenience functions, interactive help, and command completion, Recon-ng provides a powerful environment in which open source web-based reconnaissance can be conducted quickly and thoroughly."

With the help of Tim Tombes, I developed and contributed the Instagram Pushpin module to Recon-ng. It works similarly to Recon-ng's other Pushpin modules and allows you to gather Instagram posts made within a specific radius of a geolocation.

-- CHERRYPICKER

Simple command line script which spits out the contact name and security questions (if any) of a given @virginmedia.com email address.

-- IP Cam Shell

IPCS is a command line script for testing and exploiting a wide range of IP cameras as demonstrated by Craig Heffner in "Exploiting Surveillance Cameras Like a Hollywood Hacker". See the slides here: https://media.blackhat.com/us-13/US-13-Heffner-Exploiting-Network-Surveillance-Cameras-Like-A-Hollywood-Hacker-WP.pdf.
